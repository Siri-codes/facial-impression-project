# LLM Face Impressions Project

Testing whether frontier vision-language models (Gemini 2.5 Flash-Lite, GPT-5.4 Mini, Claude Sonnet 5) reproduce human first-impression judgments of faces, using the OMI dataset (Peterson et al., 2022) as ground truth.

- **Methods & findings:** see `METHODS.md`
- **Notebooks:** `notebooks/`
- **Data:** `data/` (see `data/README.md` for details)
- **Results/figures:** `results/`

Research conducted as part of a summer research project.
